This file is meant to test the wiki converters for XWiki syntax. 

Each set of syntax is shown and it's proper conversion to Confluence syntax is shown afterwards surrounded by {code} tags.


Headings

Type:	To get:
1 Heading 1	

{code}
when converted to Confluence should be this:
h1. Heading 1
{code}

1.1 Heading 2	
{code}
when converted to Confluence should be this:
h2. Heading 2
{code}

1.1.1 Heading 3	
{code}
when converted to Confluence should be this:
h3. Heading 3
{code}

1.1.1.1 Heading 4	
{code}
when converted to Confluence should be this:
h4. Heading 4
{code}

1.1.1.1.1.1 Heading 5	
{code}
when converted to Confluence should be this:
h5. Heading 5
{code}

1.1.1.1.1.1.1 Heading 6	
{code}
when converted to Confluence should be this:
h6. Heading 6
{code}

Paragraphs

Here is the first paragraph.
This text continues the same paragraph.
{code}
when converted to Confluence should be this:
h6. Heading 6
{code}


This paragraph is a new one.

Bold, Italics, Striked

a *bold* word
{code}
when converted to Confluence should be this:
a *bold* word
{code}

an __underlined__ word
{code}
when converted to Confluence should be this:
an +underlined+ word
{code}

~~text in italics~~
{code}
when converted to Confluence should be this:
_text in italics_
{code}

--striked text--
{code}
when converted to Confluence should be this:
-striked text-
{code}

Lists

* Level 1
** Level 2
*** Level 3
** Level 2
{code}
when converted to Confluence should be this:
no change
{code}	

1. Item
11.Subitem
1. Item
1*. Subitem 	
{code}
when converted to Confluence should be this:
# Item
## Subitem
# Item
#* Subitem 	
{code}

Tables

Type:	To get:
{table}
Category | Sales (K€)
Category 1 | 100
Category 2 | 50
Category 3 | 50
Total | =sum(B2:B4)
{table}

{code}
when converted to Confluence should be this:
||Category || Sales (K€)||
|Category 1 | 100|
|Category 2 | 50|
|Category 3 | 50|
|Total | =sum(B2:B4)|
{code}

Links

[WebHome]
[home>WebHome]
[Main.WebHome]
[home|Main.WebHome]
http://www.xwiki.org
[http://www.xwiki.org]
[XWiki>http://www.xwiki.org]

{code}
when converted to Confluence should be this:
[WebHome]
[home|WebHome]
[Main:WebHome]
[home|Main:WebHome]
http://www.xwiki.org
[http://www.xwiki.org]
[XWiki|http://www.xwiki.org]
{code}

Images

{image:img.png}
{image:img.png|30}
{image:img.png| |20}
{image:img.png|30|20}

{code}
when converted to Confluence should be this:
!img.png!
!img.png|width=30!
!img.png| ,length=20!
!img.png|width=30,length=20!
{code}

Attachments:

{attach:A simple text file|file=simple.txt|title=Download Simple Txt}
{attach:test.txt}

{code}
when converted to Confluence should be this:
[A simple text file|^simple.txt|title=Download Simple Txt]
[^test.txt]
{code}
